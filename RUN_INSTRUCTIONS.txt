# Healthcare RAG Medical Assistant - Run Instructions

This guide provides steps to set up and run the complete system, including data ingestion, the backend API, and the Flask web interface.

## 1. Environment Setup

Ensure you have Python 3.10+ installed. It is recommended to use a virtual environment.

```bash
# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# OR
venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt
```

## 2. Data Ingestion (First Time Only)

Before running the API, you need to download the medical datasets and create the vector store index.

```bash
python3 src/data/ingest.py
```
This will create the `data/embeddings` directory with the FAISS index.

## 3. Start the Backend API (FastAPI)

Run the backend server that handles medical query classification and RAG.

```bash
export PYTHONPATH=$PYTHONPATH:$(pwd)
python3 api/main.py
```
The API will be available at `http://localhost:8000`.

## 4. Start the Web Interface (Flask)

In a new terminal window (with the virtual environment activated), start the Flask UI.

```bash
export PYTHONPATH=$PYTHONPATH:$(pwd)
python3 ui/app.py
```
The UI will be available at `http://localhost:5000`.

## 5. Monitoring Dashboard (Optional)

You can also run the Streamlit dashboard for monitoring and analytics.

```bash
streamlit run dashboard/app.py
```

---

**Note:** Ensure that the API is running before attempting to use the Flask UI, as the UI communicates directly with the FastAPI backend.
